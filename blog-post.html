Web Geliştirmeye Başlarken: İlk Adımlar

1. Giriş

Web geliştirme yolculuğuma nasıl başladığımı ve ilk projelerimi nasıl oluşturduğumu sizlerle paylaşmak istiyorum. Bu yazıda, HTML, CSS ve JavaScript öğrenme sürecimden bahsedeceğim ve yeni başlayanlar için ipuçları vereceğim.

"Her uzman bir zamanlar acemiydi. Her profesyonel bir zamanlar amatördü. Her başarı hikayesi bir 'başlangıç' ile başlar."

2. HTML Temelleri

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan işaretleme dilidir. İlk öğrenmeniz gereken şey, HTML'in bir programlama dili olmadığı, sadece içeriği yapılandıran bir işaretleme dili olduğudur.

Temel HTML Yapısı

<!DOCTYPE html>
<html lang="tr">
<head>
    <meta charset="UTF-8">
    <title>İlk Web Sayfam</title>
</head>
<body>
    <h1>Merhaba Dünya!</h1>
    <p>Bu benim ilk web sayfam.</p>
</body>
</html>

3. CSS ile Stil Vermek

CSS (Cascading Style Sheets), HTML elementlerinize stil vermenizi sağlar. Renkler, fontlar, spacing, layout ve daha fazlası CSS ile kontrol edilir.

CSS Örneği

body {
    font-family: 'Inter', sans-serif;
    background-color: #0d1117;
    color: #f0f6fc;
}

h1 {
    color: #58a6ff;
    font-size: 2.5rem;
    margin-bottom: 1rem;
}

4. JavaScript'e Giriş

JavaScript, web sayfalarınıza interaktivite katmanızı sağlayan programlama dilidir. Dinamik içerik, kullanıcı etkileşimleri ve modern web uygulamaları JavaScript ile oluşturulur.

  • Değişkenler ve veri tipleri
  • Fonksiyonlar ve scope
  • DOM manipülasyonu
  • Event handling
  • Asenkron programlama

JavaScript Örneği

// Butona tıklandığında mesaj göster
document.getElementById('myButton').addEventListener('click', function() {
    alert('Merhaba, Web Geliştirme Dünyası!');
    console.log('Button clicked');
});

5. İlk Projeniz

İlk projeniz basit bir kişisel portfolio sitesi olabilir. İşte başlamanız için adımlar:

  1. Proje klasörünüzü oluşturun
  2. index.html dosyanızı oluşturun
  3. styles.css dosyanızı ekleyin
  4. script.js ile interaktivite katın
  5. GitHub Pages ile yayınlayın

6. Sonuç

Web geliştirme öğrenmek başta göz korkutucu görünebilir, ancak adım adım ilerleyerek ve sürekli pratik yaparak harika projeler oluşturabilirsiniz. Unutmayın, her gün yeni bir şey öğrenmek ve kodunuzu geliştirmek önemlidir.

Umarım bu yazı, web geliştirme yolculuğunuza başlamanız için size ilham vermiştir. Sorularınız varsa, benimle iletişime geçmekten çekinmeyin!
